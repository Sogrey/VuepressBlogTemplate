# **VuepressBlogTemplate** 多页面管理

[![Open in Visual Studio Code](https://open.vscode.dev/badges/open-in-vscode.svg)](https://open.vscode.dev/Sogrey/VuepressBlogTemplate)

安装依赖
``` bash
npm i
```

本地运行
``` bash
npm start
```

转pdf(需要python支持)：

``` bash
bash pdf.sh
```

编译以及提交github：

``` bash
bash deploy.sh
```
